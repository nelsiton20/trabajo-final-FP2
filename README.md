# PROYECTO: GESTIÓN DE KARDEX

El siguiente proyecto se desarrolló con el objetivo de mejorar el proceso de guardado y obtención de datos de un inventario. Se utilizó el lenguaje de programación PYTHON para crear la solución.

Este proyecto fue desarrollado colaborativamente y se siguió metodologías ágiles. Además, se agregaron test unitarios para verificar el correcto funcionamiento del programa.

## Modelos o clases
- Insumo
- Proveedor
- Movimiento
- Ingreso
- Salida
- Sistema inventario

## Pasos para ejecutar el proyecto
1. Clonar el repositorio:
```cmd
git clone <url>
cd trabajo-final-FP2
```

2. Crear entorno virtual
```
python -m venv env
```

3. Activar entorno virtual
```
env\Scripts\activate
```

4. Instalar dependencias
```
pip install -r requirements.txt
```

5. Ejecutar programa
```
python main.py
```

6. Ejecutar tests
```
pytest
```
